The Story

If we're going to move critical software infrastructure to memory safe code we're going to need to make some investments in tools. It needs to be as easy as possible to build safer software.

What We've Done

  1. We contracted with Rémy Rakic to spend six months improving Rust compile times, from January to September of 2022.
  2. We have signed a six-month contract with Ferrous Systems to make various improvements to the bindgen tool. Work will begin within the next few months.

What's Next

We plan to extend Rémy Rakic's contract to improve Rust compile times by an additional six months.
