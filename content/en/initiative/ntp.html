The Story

Keeping track of time is critical for servers and clients on the Internet. NTP is how the Internet keeps track of time. Since NTP implementations are so critical and they're on the network, it's important that they be secure.

The most popular NTP implementations today are written in C and thus not memory safe. We're aiming to fix that.

What We've Done

  • Milestone 1 Complete: The NTP client is ready for use! This work was done by Tweede golf under contract with ISRG's Prossimo project.
  • Milestone 2 Complete: The NTP server is ready for use! This work was done by Tweede golf under contract with ISRG's Prossimo project.
  • Milestone 3 Complete: Network Time Security (NTS) is fully implemented.
  • Milestone 4 Complete: A third party security audit performed by Radically Open Security was completed.

What's Next

The software is ready for production use and Prossimo's work here is done. Tweede Golf is the new home for ntpd-rs, we thank them for all of their great work!
